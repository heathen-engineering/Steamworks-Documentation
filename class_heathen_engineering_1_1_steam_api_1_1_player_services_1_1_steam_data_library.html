Detailed Description

A collection of fields that can be serialized and saved to the Steam Remote Storage system. This can be most simply thought of as the structure of a save file.

To create a new SteamDataLibrary in your project locate the folder where you want it to live and right click selecting the following path

Create >> Library >> Steam Game Data Library

This will create a new SteamDataLibrary in your project where you add fields

To add a field locate a folder where you want the field to reside, right click and select

Create >> Variables

And then select the type of field you would like to create. These HeathenEngineering.Scriptable.DataVariable

◆ filePrefix

string HeathenEngineering.SteamApi.PlayerServices.SteamDataLibrary.filePrefix

The prefix of the file.

This serves a funciton similar to a file extension in Windows in that the prefix of a file is how the system determins which SteamDataLibrary a file belongs to ... e.g. if the prefix of a library is 'sys_' then all files whoes name start with sys_ will be assumed to belong to that library.
