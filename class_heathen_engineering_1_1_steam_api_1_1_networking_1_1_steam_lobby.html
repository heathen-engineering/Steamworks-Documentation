◆ SetGameServer() [1/4]

void HeathenEngineering.SteamApi.Networking.SteamLobby.SetGameServer ( )

This overload uses the lobby owner's CSteamID as the server ID which is typical of P2P session.

Sets the game server associated with the lobby. This can only be set by the owner of the lobby. Either the IP/Port or the Steam ID of the game server must be valid, depending on how you want the clients to be able to connect. A LobbyGameCreated_t callback will be sent to all players in the lobby, usually at this point, the users will join the specified game server.

See SetLobbyGameServer

in Valve's documentaiton. This will update the game server settings for the lobby notifying all members of the new data. This should only be called when you are ready for the members to join the server.

◆ GameServer

LobbyGameServerInformation HeathenEngineering.SteamApi.Networking.SteamLobby.GameServer
get

The game server information stored against the lobby

This data is set when the host calls SetLobbyGameServer or one of its variants. Uppon calling SetLobbyGameServer the Valve backend will raise OnGameServerSet for all members other than the host the paramiter of which also contains server data. The typical use case of this field is when a member has join a persistent lobby after the game server has been started.

bool HeathenEngineering.SteamApi.Networking.SteamLobby.HasGameServer

Does this lobby have a game server registered to it

Calls SteamMatchmaking.GetLobbyGameServer(CSteamID, out uint, out ushort, out CSteamID) and cashes the data to GameServer. It is not usually nessisary to check this value since the set game server callback from Steam will automatically update these values if the user was connected to the lobby when the set game server data was called.

◆ IsHost

bool HeathenEngineering.SteamApi.Networking.SteamLobby.IsHost
get

Is the user the host of this lobby

Calls SteamMatchmaking.GetLobbyOwner(CSteamID) and compares the results to SteamUser.GetSteamID(). This returns true if the provided lobby ID is a legitimate ID and if Valve indicates that the lobby has members and if the owner of the lobby is the current player.

◆ Joinable

bool HeathenEngineering.SteamApi.Networking.SteamLobby.Joinable
getset

Sets whether or not a lobby is joinable by other players. This always defaults to enabled for a new lobby. If joining is disabled, then no players can join, even if they are a friend or have been invited. Lobbies with joining disabled will not be returned from a lobby search.

This is only accruate on the lobby owner's system and may not be accurate if the owner has changed since it was set. In general its advisable that you have a new owner set Joinable to the desired value when they are made owner doing so will cause the value to sync for that user.
