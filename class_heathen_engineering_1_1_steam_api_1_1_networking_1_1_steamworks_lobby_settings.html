Detailed Description

Handles configuration and tracking for Steam Lobby

Steam allows users to be in 1 'regular' lobby and up to 2 'invisible' lobbies. This system simply manages a list of lobbies and doesn't try to enforce Valve's rule for 1 regular and 2 invisible. It is up to the indavidual game developer to handle lobby types and enforce rules regarding how many lobbies a user can be in.
