Detailed Description

Presents log messages to a text object and manages a parent scroll rect simulating the behaviour of the Uniy console panel for use at run time.

For example: calling Debug.Log(string message) will cause the message to be appended to the Console.text field.

You can find a working example of this componenet in every example scene.
