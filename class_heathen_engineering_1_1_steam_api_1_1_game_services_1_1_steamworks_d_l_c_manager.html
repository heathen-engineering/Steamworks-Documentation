Detailed Description

Links and manages SteamDLCData objects.

This behaviour handles updating the status of each SteamDLCData object.

Steam DLC or Downloadable Content is defined on the Steam API in your Steam Portal. Please carfully read https://partner.steamgames.com/doc/store/application/dlc before designing features are this concept.
