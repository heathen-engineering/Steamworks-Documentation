Detailed Description

Demonstrates Achievement and Status update activity

This is a demo script and is not inteded for prodcution use. It does not adhere to best practices for production grade code instead it focuses on verbose demonstration of easily understood features and fucntons. This code should be used to understand how one can use the members of the Steam Achievement and Steam Stat related objects.
