Detailed Description

This is a simple example of how to create a custom InventoryItemDefinition object for your game.

In most case you would add additional fields so your game client can render rich information about an item such as textures, models, names, descriptions, etc. The InventoryItemDefinition base class provides only the basics that the SteamworksInventoryManager needs to operate on the item.
