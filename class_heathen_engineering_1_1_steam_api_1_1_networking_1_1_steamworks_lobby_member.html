◆ Metadata

SteamworksLobbyMetadata HeathenEngineering.SteamApi.Networking.SteamworksLobbyMember.Metadata
get

No longer used.

Use of this member will throw an error on compilation.

To set or get metadata values for a member use the string indexer e.g.

myLobbyMember["metadataKey"] = "This sets the value of the key metadataKey.";

Valve does not provide a means to fetch all set values nor iterate over values by index for member metadata as it does for Lobby metadata. There for you must know the key of the data you are looking for. Attempting to get a field that doesn't exist simply results in an empty string and will not cause an out of bounds exception.

The previous system had you set possible metadata keys on the LobbySettings object and used that to assume keys available on the member. As the system now supports multiple lobbies at a time this is no longer appropreat on the lobby settings object. You can create your own index of possible keys and use the indexer provided here to access the values.
