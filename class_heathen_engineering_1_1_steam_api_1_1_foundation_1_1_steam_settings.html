Detailed Description

The root of Heathen Engieering's Steamworks system. SteamSettings provides access to all core funcitonality including stats, achievements, the friend system and the overlay system.

The SteamSettings object is the root of Heathen Engineering's Steamworks kit. SteamSettings contains the configuration for the fundamental systems of the Steam API and provides access to all core funcitonality. You can easily access the active SteamSettings object any time via current a static member that is populated on initalization of the Steam API with the settings that are being used to configure it.

SteamSettings is divided into 2 major areas being client and server. The client member provides easy access to features and systems relivent for your "client" that is the applicaiton the end user is actually playing e.g. your game. This would include features such as overlay, friends, clans, stats, achievements, etc. server in contrast deals with tthe configuraiton of Steam Game Server features and only comes into play for server builds. Note that the server member and its funcitonality are stripped out of client builds, that is it is only accessable in a server build and in the Unity Editor

bool HeathenEngineering.SteamApi.Foundation.SteamSettings.Initialized
get

Indicates rather or not the Steamworks API is initalized

This value gets set to true when Init is called by the SteamworksFoundationManager. Note that if Steam API fails to initalize such as if the Steam client is not installed, running and logged in with a valid Steam user then the call to Init will fail and the Initialized value will remain false.
