Detailed Description

The root of Heathen Engieering's Steamworks system. HeathenEngineering.SteamApi.Foundation.SteamSettings provides access to all core funcitonality including stats, achievements, the friend system and the overlay system.

This object defines your application's ID, your local user's HeathenEngineering.SteamApi.Foundation.SteamUserData object and manages the lists of all known users.
